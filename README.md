# 🏗️ SysDesign

An interactive, visual platform designed to help developers build, understand, and master system design concepts through hands-on exploration.

---

## 🚀 Features

* **Interactive Canvases:** Drag-and-drop components (load balancers, databases, message queues) to visualize complex architectures.
* **Real-time Feedback:** Instantly see bottlenecks, latency estimates, and failure points in your design.
* **Scenario Testing:** Simulate traffic spikes and region failures to test system resilience.
* **Architecture Templates:** Start with pre-built templates for common use cases (e.g., URL shorteners, chat apps, video streaming).
* **AI-Assisted Insights:** Get automated suggestions for optimizing your architecture for scale and cost.

## 🛠️ Tech Stack

* **Framework:** Next.js
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Package Manager:** Bun / npm


---

## 💻 Getting Started

To run the development server locally, use your preferred package manager:

Using **Bun** (Recommended):
```bash
bun run dev
    or
npm run dev
```
## Open
Open http://localhost:3000 in your browser to see the application.
